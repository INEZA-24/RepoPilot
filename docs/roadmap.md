# RepoPilot Roadmap

## Implemented

### Phase 1 MVP

- Repository metadata
- Languages
- README preview
- Open issues
- Basic contributor dashboard

### Phase 2A: NVIDIA Nemotron Contribution Entry Points

- Optional user-triggered AI recommendations after repository analysis
- Contributor profile inputs for experience, skills, and preferred contribution type
- Evidence gathering from issues, filtered paths, contributor docs, and manifests
- Deterministic issue ranking before AI usage
- Prompt-injection protection and server-side verification
- Heuristic fallback labeled as `heuristic-fallback`

## Planned, Not Implemented

### Later Phase 2

- Broader AI repository summary
- Recommended reading order
- Beginner friendliness analysis
- Learning path

### Phase 3

- Contribution roadmap beyond entry-point suggestions
- Contributor readiness score
- Repository health score
- Maintainer recommendations

## Guardrails

RepoPilot should not become GitHub search, a generic repository chat app, an automated coding agent, or a vanity analytics dashboard. Every roadmap item must reduce contributor onboarding time.
